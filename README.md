## 白噪音助眠到底有没有用？潮汐的逻辑在这里

说白了，白噪音之所以能助眠，是因为它掩盖了环境杂音。你睡不着，大部分时候不是因为太安静，而是因为脑子停不下来——邻居说话声、街上的车声、自己的思绪——这些"信号"都在拉着你的注意力。白噪音制造一个稳定的声音背景，把这些干扰盖住。

潮汐提供平衡呼吸（5秒吸气+5秒呼气）和 4-7-8 呼吸两种科学呼吸法。4-7-8 呼吸由美国科学家 Dr. Andrew Weil 提出，4 秒吸气、7 秒屏气、8 秒呼气，有助于放松身心并帮助快速入眠。

换气时潮汐不用人声提示，而是用区别于背景音的特殊音效，避免分心。这个细节挺聪明的。

潮汐官网收录了不同媒体的真实评价：《今日 App》（App Store）称其"让人感觉像是进入另一个世界，没有一丝浮躁"；少数派认为"素雅而安静的界面也是我想打开它的理由之一"；《流动日报》写到"听着潮汐的声音，莫名地就静下心来"。
